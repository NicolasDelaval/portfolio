A propos de moi

« Je suis junior, il est vrai ; mais aux développeurs motivés, la valeur ajoutée n'attend point le nombre des années ».

Ancien formateur progiciel, j’ai pris la décision de devenir développeur web. Ce choix est le fruit d’une mûre réflexion et la somme de plusieurs éléments de motivation :

  • Mon attrait pour le web et ma volonté d’être un de ses acteurs.
  • L’envie d’exercer un métier où constance des évolutions technologiques et curiosité, autoformation vont de pair.
  • Allier autonomie et travail en équipe.
  • Le plaisir de rendre fonctionnelle une maquette d’un designer, de faire évoluer un produit en tenant compte d’un cahier des charges.

A la recherche d’une première expérience dans le développement, je vous propose de mettre l’ensemble de mes compétences techniques et de mes savoir être à votre service.

Application Front End: Réservation de vélos


Cette application purement front-end de réservation de vélos fut réalisée durant ma formation Openclassrooms.
L'objectif premier de ce projet était la découverte et l'apprentissage de JavaScript.

  • Fonctionnalités:

  • Tutoriel sous la forme d'un carousel.
  • Possibilité d'afficher ou de masquer à loisir le tutoriel.
  • A partir de la carte, possibilité de sélectionner une station et de connaître le nombre de vélos disponibles.
  • Une fois la station sélectionnée, possibilité de réserver un vélo via un formulaire simple.
  • L'utilisateur valide sa réservation en signant sur un canvas.
  • La réservation effectuée, un décompte s'affiche : celle-ci est valide pendant vingt minutes.

  • API / Compétences / Langages utilisés:

  • API JCDecaux
  • API Google Map
  • CSS3
  • HTML 5
  • Javascript ECMAScript 5 POO
  • jQuery
  • Session Store / Local Store

Moteur de Blog Php [C.R.U.D]


Ce moteur de blog Php fut réalisé durant ma formation Openclassrooms.
L'objectif premier de ce projet était la découverte et l'apprentissage de Php.

  • Fonctionnalités:

  • L'administrateur peut ajouter, modifier, supprimer un billet.
  • L'administrateur peut modérer les commentaires laissés sur les billets : les approuver ou les supprimer
  • S'il est enregistré et connecté, un utilisateur lecteur peut commenter un billet du blog.
  • S'il est enregistré et connecté, un utilisateur lecteur peut signaler un commentaire.

  • Technologies et Langages utilisés:

  • Bootstrap 3
  • CSS3
  • HTML 5
  • Modèle MVC
  • MySQL
  • Php [syntaxe POO pour les models]

Blog Musical- Annuaire de Disquaires [C.R.U.D]


Ce blog musical- annuaire de disquaires fut réalisé comme projet libre durant ma formation Openclassrooms.

  • Fonctionnalités:

  • L'administrateur peut écrire, modifier, supprimer un billet sur le blog.
  • L'administrateur peut ajouter, modifier, supprimer un disquaire sur la carte du site.
  • Un utilisateur peut lire les billets du blog et consulter la carte des disquaires sans être enregistré ou connecté.
  • S'il est enregistré et connecté, un utilisateur lecteur peut commenter un billet du blog.

  • API / Compétences / Langages utilisés:

  • Bootstrap 3
  • CSS3
  • HTML 5
  • Javascript ECMAScript 5 POO
